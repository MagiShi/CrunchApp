Home Page and Searching

When you first arrive at the Home page, you see a list of all the items in the system. You can narrow down the list by using the Filter Menu. You can also search for a specific item by typing its name or its unique barcode into the search bar.

Filter Menu

You can access the Filter Menu by clicking on the Filter Menu icon (  ) next to the search bar.

Open up the different panels (e.g. "Color," "Size") to specify which characteristics you wish to narrow down the list by. The default state, where all the checkboxes under each panel are unchecked, means the list is not filtered by any of those characteristics. For example, if the Size panel has no checkboxes checked, then you are viewing items of all sizes.

After you check the appropriate checkboxes, make sure to press the Apply Filters button in order to refresh the list of items with the filters now applied.
